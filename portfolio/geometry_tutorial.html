Geometry Tutorial

Web application running as an Interactive Geometry tutorial

For my senior project in 2010, which I carried out with two classmates, we developed a web application running as an interactive tutorial to help freshmen in studying Geometry for the admission tests of the Technological University of Panama.

Our first option was creating the tutorial with Flash, but given the drawbacks of Flash, we decided to create a web application using jQuery, HTML and used Flash only for the content animation, such as dinamyc examples.

This was my first web project, when I realized that I liked web development and I wanted to pursue a career in this field.

Responsibilities

  • Created and developed single page application using jQuery, AJAX, HTML, CSS and PHP
  • Integrated Flash animation for every screen with jQuery
  • Used PHP with jQuery and AJAX to load the content dinamically
  • Created animation with jQuery
  • Worked in the Login module, implementing functionality with PHP, MySql, jQuery and JavaScript
  • Created tests with jQuery and validated using PHP functions
