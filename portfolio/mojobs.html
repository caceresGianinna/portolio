Jobs MOGov mobile application

I had the pleasure to work as a front-end developer contractor, for the Missouri Government client.JobsMoGov is an hybrid mobile application created with Ionic Framework, Angularjs, HTML5/CSS3, and Apache Cordova, for iOS and Android systems.

Jobs.mo.gov mobile application is the mobile version of the web portal for information and on-line services provided by the Division of Workface Development.

The application provides workforce services for businesses and job seekers in Missouri, helping job seekers find gainful employment and providing businesses with a skilled, trained workforce.

Technologies Used:
  • HTML5/CSS3
  • Ionic open source front-end SDK for developing hybrid mobile apps with HTML5
  • Apache Cordoba
  • AngularJs
  • Sublime Editor
  • Github

Some of my responsibilities were:

  • Created mobile application layout and front-end using HTML5/CSS3, and Ionic mobile framework components.
  • Implemented Apache Cordova plugins such as InAppBrowser, and Geolocation to get user's position from mobile device.
  • Used JSON to get data from back end, consuming Restufl web services in CORS environment.
  • Implemented Favorites module using HTML WebSql and Local Storage.
  • Utilized Google Maps API with AngularJs for map functionalities.
  • Created the application theme and interactions with CSS3 and Ionic CSS components.
  • Worked with Google Maps Library and Apache Cordova Geolocation Plugin to get user's zip code based on user's location.
  • Used AngularJs animations with CSS classes.

Environment: Windows 7, Git
